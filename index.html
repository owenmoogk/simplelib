SimpleLib

Welcome to SimpleLib. The python library to make everything a bit easier. Entirely open source (obviously). Enjoy.

About

This project was made mostly for fun, and to learn about complex data structures used in CS and software engineering. It is designed to provide functions that ease handling binary trees, linked lists, arrays, and more.

Hashmaps

The hashmap functions were implemented into SimpleLib for users to have a bit more access to the hashmap than a python dictionary would allow. But in most cases, a dictionary is the best choice. Syntax is equivalent to a dictionary.

Example:

t["march 6"] = 130
t["march 6"] = 20
t["april 2"] = "45 degrees"
t["april 3"] = "23 degrees"
print(t["march 6"])
print(t["april 2"])

Output:

130
45 degrees

Binary Search Trees

Binary search trees are useful for data storage, implementing a tree like structure. They are sorted with left and right tree branches, increasing and decreasing respectivley. SimpleLib is compatible with both numerical and string search trees.

Function Description

Example:

numbers = [8,56,13,4,3,5,6,7,-9,155,155,155]
numbersTree = buildTree(numbers, 7)
numbersTree.addChild(-14)
numbersTree.display()
print(numbersTree.rightSideView())

Output:

           7        
         /   \       
        4    8_    
      / \        \   
      3 5       56_  
      /   \      /      \
  -9   6    13    155
  /
-14

[7, 8, 56, 155]

Linked Lists

Linked lists are a data structure similar to arrays / lists, but are superior in some cases. Instead of items having indexes, they are all linked to eachother, leading to quicker 'big O' time for many operations. SimpleLib is compatible with both singly linked lists and doubly linked lists. At this point they both employ the same functions.

Function Description

Example:

mylist = singly_linked_list.singlyLinkedList()
mylist.append(1)
mylist.append(6)
mylist.display()
mylist.erase(1)
mylist.insertAfter("hello",1)
mylist.display()
print("Size:",mylist.size())
print("'hello' index:",mylist.getItemIndex("hello"))

Output:

[1, 6]
[1, 'hello']
Size: 2
'hello' index: 1
